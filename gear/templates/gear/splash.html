Gear Calculator

Ever wonder how much grinding you have to do to get every hero to Orange +4? Brace yourself. It's a lot. But this tool can help you decide which scraps you need to hang onto and which ones can be sold or used for enchanting. Just update each of the heroes to reflect that items you've already equipped (including items you've already sacrificed for Orange Legendary Quests.

Items Catalog

Need to look up info on any given item, scrap, scroll, etc.? Like where to get it, what's it used for, who can equip it, etc.? Look no further. We have plans to add more data here in the future like how much it is worth when enchanting.

Grind Optimizer

Want to know where the best place is to get all the gear you currently need to complete specific heroes' current rarities? Pick the heroes you want to focus on and then input how many of the necessary scraps you need to complete their current rarities, hit "Optimize My Grind" and it will tell you which stages have the most efficient path forward for you. Note that marking gear complete for a given hero on either this tool or the Gear Calculator will effect both tools!

Runes Optimizer

If you've ever needed help deciding which hero should get which rune, this tool will be for you. Simply quickly select which stats and what type of rune you have and we will show you which heroes would most benefit from that rune.
